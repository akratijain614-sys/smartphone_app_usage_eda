## Smartphone App Usage and Market Trends Analysis
This project analyzes Google Play Store app data to understand patterns in app adoption, user engagement, and monetization strategies.

## Dataset
Google Play Store Apps dataset from Kaggle.

## Objectives
- Identify app categories that dominate downloads
- Analyze the relationship between ratings and installs
- Compare free and paid apps in terms of user satisfaction
- Explore app adoption using interactive visualizations
## Methodology
Exploratory Data Analysis (EDA) was performed using Python. Data cleaning was applied to convert text-based fields into numeric formats. Descriptive and
interactive analyses were used to answer business questions.

## Tools Used
- Python (pandas, matplotlib, plotly, ipywidgets)
- Google colab
- Git & GitHub
- 
## Output
An interactive analysis showing the top 5 apps by total installs for selected categories and monetization types.
